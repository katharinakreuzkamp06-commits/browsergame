# Webentwicklung

**Digitales Bücherregal**

**Beschreibung**
Bei meinem Projekt geht es um eine Webseite zum Überblick über meine Bücher. Wenn man die Webseite aufruft, ist dort ein Regal zu sehen und unten befindet sich eine Leiste mit verschiedenen Buttons. Oben links werden die jeweiligen Anzahlen (die unten bei Funktionen genannt werden) eingeblendet.

**Funktionen**
* Bücher können mit Titel hinzugefügt werden
* Ein neu hinzugefügtes Buch wird im Regal gezeigt
* Bücher können gelöscht werden
* Daraufhin wird der Platz im Regal auch wieder frei
* Es wird angegeben ob das Buch bei gelesen oder zu lesen gezählt werden soll
* Diese Angabe kann später wieder geändert werden (nur von ungelesen zu gelesen)
* Speichert : die Bücher an sich, die Anzahl der Bücher, die Anzahl gelesner Bücher, die Anzahl zu lesender Bücher

**Seitenstruktur**
* Auf der Homepage befindet sich das Regal, mit den Scores oben links und den Buttons unten
* Der Button für Buch hinzufügen öffnet ein Popup Fenster in dem der Name eingefügt werden kann und angekreuzt wird ob das Buch bereits gelesen wurde oder nicht
* Beim Button für Buch löschen kommt ebenfalls ein Fenster, wo mann den Namen eingibt, daraufhin wird das Buch mit übereinstimmendem Namen gelöscht
* Ein Button Status ändern verlinkt einen zu einer Seite, wo die Namen der eingetragenen Bücher gelistet sind und man diese anklicken kann, um sie von zu lesen auf gelesen zu setzen oder man tut dies ebenfalls über ein Fenster
* Ein Button namens Speichern kann angeklickt werden um den aktuellen Stand zu speichern
* Button weiteres Regal verlinkt einen zu einer zweiten Seite, die der ersten ähnelt jedoch gibt es dort einen Button um zur ersten Seite zurück zugelangen und die Anzahlen dort werden mit der von der ersten Seite addiert ( die gesammten Daten der ersten Seite sind weiterhin bei den Anzahlen vorhanden und veränderbar)
